# Transformer Architecture and Modern NLP Models

A comprehensive educational repository that explains the architecture, workflow, and internal components of modern Transformer-based language models. This project provides detailed documentation of the Transformer architecture and its evolution through models such as **DeBERTa-v3 Small**, **SmolLM2**, and **Longformer Encoder-Decoder (LED)**.

The goal of this repository is to help students, researchers, and AI engineers understand how modern NLP models work from the architectural level.

---

## Project Overview

This repository covers:

- Transformer Architecture
- Self-Attention Mechanism
- Masked (Causal) Attention
- Multi-Head Attention (MHA)
- Multi-Query Attention (MQA)
- Grouped Query Attention (GQA)
- Flash Attention
- BERT Architecture
- DeBERTa-v3 Small
- SmolLM2
- Longformer Encoder-Decoder (LED)

Each topic is explained with:
- Architecture diagrams
- Workflow diagrams
- Step-by-step explanations
- Component descriptions
- Advantages and disadvantages
- Model comparisons
- Technical documentation
